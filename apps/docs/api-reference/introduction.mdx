# API Reference

The Agentset API provides a comprehensive set of endpoints for managing namespaces, documents, and ingest jobs.

## Authentication

All API endpoints require authentication using an API key. Learn more in the [Authentication](/api-reference/authentication) section.

## Base URL

The base URL for all API endpoints is:

```
https://api.agentset.ai
```

## API Endpoints

The API is organized around the following resources:

- **Namespaces**: Create and manage isolated data environments
- **Documents**: Retrieve and manage document data
- **Ingest Jobs**: Upload and process content

Endpoints are defined according to the OpenAPI specification. You can explore each endpoint for detailed request and response examples.
