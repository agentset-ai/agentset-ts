# Namespaces

Namespaces in Agentset provide a way to organize and isolate your data. Each namespace represents a separate collection of documents that can be searched and used for chat independently.

## Key Features

- **Isolation**: Keep different document sets separate
- **Organization**: Group related documents together
- **Efficiency**: Search within specific subsets of your data

## Available Operations

| Operation | Description |
|-----------|-------------|
| [List](/api-reference/namespaces/list) | Retrieve all namespaces in your account |
| [Get](/api-reference/namespaces/get) | Retrieve details of a specific namespace |
| [Create](/api-reference/namespaces/create) | Create a new namespace |
| [Update](/api-reference/namespaces/update) | Modify an existing namespace |
| [Delete](/api-reference/namespaces/delete) | Remove a namespace and its contents |

## Getting Started

To begin working with namespaces, create your first namespace using the [Create a Namespace](/api-reference/namespaces/create) endpoint.
